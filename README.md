**Enterprise RAG Assistant**

An end-to-end Retrieval-Augmented Generation (RAG) system being built from scratch to understand the architecture and implementation of production AI assistants.

The project covers the complete retrieval pipeline:

Document ingestion
PDF processing
Chunking and metadata extraction
Embedding generation
Vector search with FAISS
Reranking
Prompt construction
LLM orchestration

The goal is not just to use existing frameworks, but to understand the design decisions, tradeoffs, and system architecture behind enterprise AI applications.
